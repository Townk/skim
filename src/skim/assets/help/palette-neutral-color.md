# Neutral Color

Each layer in the keymap images generated by SKIM has a _main color_
associated with them. In the layout design used by SKIM, there are some keys in
the thumb clusters that use a neutral color to distinguish them from the rest.

This neutral color field defines the color used in these thumb keys.

**Default:** `#6F768B`

Accepts CSS color names (e.g. `slategray`) or hex values (e.g. `#6F768B`).
Autocomplete is available for named colors.
