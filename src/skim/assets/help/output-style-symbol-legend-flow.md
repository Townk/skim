# Symbol legend flow

Controls how entries flow through the multi-column symbol legend below the
keymap.

- **Column-major** (default): entries fill the leftmost column top-to-bottom,
  then continue at the top of the next column. Reading order is "down then
  right".
- **Row-major**: entries fill the top row left-to-right, then drop to the
  next row. Reading order is "right then down".

Both modes show the same set of entries; only the visual order changes.
