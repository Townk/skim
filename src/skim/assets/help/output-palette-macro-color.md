# Macro Color

Color used for macro badges and the macro-table title block in the
rendered keymap image. Marks macro-related elements with a consistent
hue so they're easy to spot at a glance.

**Default:** `#89511C`

Accepts CSS color names (e.g. `sienna`) or hex values (e.g. `#89511C`).
Autocomplete is available for named colors.
